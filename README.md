# cardiac-prediction
A Python-based ML project for Cardiac Disease prediction using classification algorithms
## Cardiac Disease Prediction using Machine Learning

## Overview

This project develops a machine learning model to predict cardiac disease using patient clinical data. Data preprocessing and feature scaling were performed before training the models.

## Models Evaluated

- Decision Tree
- Random Forest
- K-Nearest Neighbors (KNN)
- Support Vector Machine (SVM)

Among these, *Support Vector Machine (SVM)* achieved the best performance and was selected as the final model.

## Key Result

- *Recall (Positive Class): 0.97*

A high recall is crucial in medical diagnosis because it minimizes false negatives, reducing the risk of incorrectly classifying a patient with cardiac disease as healthy.
